Nome do Sistema: SIGD (Sistema de Gerenciamento de Projetos de Graduação).

Versão: beta 1

Sobre: Primordialmente é importante ressaltar que o objetivo do trabalho realizado em parceria com o professor Patrick Letouze é criar uma plataforma web 2.0 que facilite não só o gerenciamento de projetos de graduação bem como a gerencia de disciplinas como metodologia cientifica. Inicialmente no que tange as instancias da UFT para que depois se atinja outras regiões até que se alcance nível Brasil por completo, sendo essa a meta deste sistema.
